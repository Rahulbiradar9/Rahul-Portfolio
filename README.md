# Rahul's Portfolio Website

A premium, minimalist, and interactive personal portfolio website showcasing my experience, projects, and skills as a Software Engineer. Built with modern web technologies and designed with a clean, modern aesthetic.

## 🚀 Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/) (App Directory)
- **Library:** [React 19](https://react.dev/)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Smooth Scrolling:** [@studio-freight/react-lenis](https://github.com/studio-freight/react-lenis)
- **Icons:** [Lucide React](https://lucide.dev/) & [React Icons](https://react-icons.github.io/react-icons/)
- **Interactive Elements:** Magnetic buttons, Custom cursor, Scroll progress bar, AI Chatbot integration

## ✨ Features

- **Minimalist Aesthetic:** Clean, modern, "boxless" editorial design.
- **Smooth Animations:** Fluid page transitions and scroll-based reveal effects powered by Framer Motion.
- **Interactive UI:** Features a custom cursor and magnetic button effects for enhanced user experience.
- **Responsive Design:** Fully optimized for mobile, tablet, and desktop viewing.
- **Comprehensive Sections:** 
  - **About & Skills:** Technical expertise and background.
  - **Experience & Projects:** Timeline of professional roles and real-world engineering work.
  - **Contact & Chatbot:** Seamless ways to get in touch or interact.

## 🛠 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Project Structure

- `src/app/` - Next.js app router pages, global css, and layouts.
- `src/components/` - Reusable React components (Hero, Projects, Experience, CustomCursor, etc.).
- `public/` - Static assets including images and icons.
